# Requirements
- Neovim 0.8
- 'curl' command
- gcc or other C compiler
- git

# Quickstart
- **Windows**: git clone https://github.com/luxgile/nvim_config.git $HOME/AppData/Local/nvim/
- **Linux**: git clone https://github.com/luxgile/nvim_config.git ~/.config/nvim/
- Open nvim and run:
    - :PlugInstall
    - :PackerInstall
    - :PackerUpdate

# Keysbindings
- F10 - Open settings file
